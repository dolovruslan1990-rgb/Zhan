# Weather Diary

**Автор:** Жангериев Азамат

## Краткое описание

Weather Diary — это десктопное приложение для ведения дневника погоды с графическим интерфейсом. Программа позволяет:
* добавлять записи о погоде с указанием даты, температуры, описания и наличия осадков;
* просматривать все сохранённые записи в табличном виде;
* фильтровать записи по дате и температуре;
* сохранять данные в формате JSON и загружать их из файла.

Приложение разработано на языке Python с использованием встроенной библиотеки `tkinter` для создания GUI.

## Функциональные возможности

* **Добавление записей:** ввод данных о погоде через удобный интерфейс.
* **Валидация данных:** проверка корректности формата даты, числового значения температуры и обязательности заполнения описания.
* **Просмотр записей:** отображение всех записей в виде таблицы.
* **Фильтрация:** возможность отфильтровать записи:
    * по конкретной дате;
    * по температуре (показать записи с температурой выше указанного значения).
* **Сохранение и загрузка:** экспорт данных в файл `weather_data.json` и импорт из него.
* **Сброс фильтров:** очистка условий фильтрации и возврат к полному списку записей.
